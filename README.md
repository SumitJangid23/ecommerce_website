#E-Commerce Website

A fully functional frontend E-Commerce Website built using HTML, CSS, and JavaScript.  
This project demonstrates core e-commerce features with a clean and responsive user interface.

---

Features

- Product listing page  
- Product detail page  
- Add to cart functionality  
- Dynamic cart with quantity update  
- Total price calculation  
- Live search filtering  
- Checkout system with multiple payment options (UPI, Card, COD)  
- Responsive and user-friendly UI  


Tech Stack

- HTML  
- CSS  
- JavaScript  


What I Learned

- DOM manipulation  
- JavaScript event handling  
- Building real-world UI  
- Managing cart state using localStorage  
- Structuring a multi-page web project  


 Future Improvements

- Wishlist functionality  
- User authentication  
- Backend integration  
- Payment gateway integration
